# Thin9-Front
Repository of Thin9 Frontend


## Stack
- React: 5.0.1
- node: v18.17.0
- yarn 1.22.19

## Install with Docker 

Run at project root directory.

```shell
docker-compose up -d
```


## Instally
yarn

yarn add @nivo/line axios dotenv bootstrap react-bootstrap react-router-dom web-vitals

## Run Server


```shell
yarn start
```

## Service Port
- Frontend: 3000
